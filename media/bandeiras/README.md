# Observação sobre essa pasta bandeiras/

Se você olhar o model de Times, verá que tem um campo de tipo imagem chamado "bandeira".
A ideia, inicialmente, era que eu utilizasse as bandeiras dos países no app, porém durante
o desenvolvimento acabei, POR HORA, descartando essa ideia, mas mantive o campo lá e também essa pasta, para caso no futuro eu queira adicionar no app as bandeiras das seleções, bastará ir no admin e adicionar as bandeiras nos times, que elas aparecerão nesta pasta.
